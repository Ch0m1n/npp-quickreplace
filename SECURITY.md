# Security policy

NppQuickReplace is designed to work entirely offline. It does not transmit document text,
replacement rules, file paths, clipboard data, or usage data.

Please report a suspected vulnerability privately through GitHub Security Advisories rather
than opening a public issue. Include the affected version, Notepad++ version, reproduction
steps, and whether a crafted document or replacement file is required.

The `replacements.json` file is data only. Its contents are never executed as code.
