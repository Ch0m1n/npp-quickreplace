# Architecture

## 목표

NppQuickReplace 0.5 알파는 입력 지연과 데이터 손실 없이 구분 문자·즉시 치환을 제공하고, JSON이나 CSV를 직접 고치지 않아도 되는 네이티브 규칙·그룹 관리 흐름을 제공해요.

## 입력 흐름

```text
SCN_CHARADDED / 선택적 SCN_MODIFIED 붙여넣기
  -> IME tentative 입력, 전체/문서 상태, 읽기 전용, 다중 선택 확인
  -> 구분 문자 또는 immediate 후보 분류
  -> 버퍼별 캐시에서 파일 확장자를 조회하고 전체 경로·Notepad++ 언어 문맥 수집
  -> 커서 앞 제한 범위를 SCI_GETTEXTRANGEFULL 한 번으로 읽기
  -> Scintilla 코드 페이지에서 UTF-16으로 변환하고 Unicode 경계 탐색
  -> NFC 정규화 후 RuleStore 해시 인덱스와 확장자·경로 글롭·언어 필터 조회
  -> 결과를 현재 문서 코드 페이지로 안전하게 변환
  -> RAII Undo + 내부 편집 가드에서 SCI_REPLACETARGET
```

표현할 수 없는 문자를 ANSI/DBCS 문서에 넣어야 한다면 치환을 건너뛰고 로그에 경고해요. 플러그인은 문서 인코딩을 임의로 바꾸지 않아요.

## 컴포넌트

### RuleStore

- JSON을 임시 구조로 전부 검증한 뒤 성공할 때만 현재 인덱스와 교체해요.
- 트리거를 Unicode NFC로 정규화하고, 영문 ASCII 대소문자 미구분 인덱스를 별도로 유지해요.
- `immediate` 규칙을 지원하고 더 긴 트리거를 가로채는 접두어 충돌은 거부해요.
- 스키마 버전, 그룹/규칙 ID, activation, 중복, 접두어를 검사하고 경고를 반환해요.
- 64 MiB 파일, 100,000개 규칙, 128 MiB 전체 텍스트 한도를 적용해요.
- 확장자, ASCII 대소문자 비구분 전체 경로 글롭, Notepad++ 언어 이름 필터를 함께 검사해요.

### ConfigStore

- 기존 JSON의 알 수 없는 필드를 보존하면서 알려진 필드만 갱신해요.
- 고유 임시 파일에 기록하고 `FlushFileBuffers` 후 `ReplaceFileW` 또는 `MoveFileExW`로 교체해요.
- 프로세스 간 이름 있는 mutex로 설정 쓰기를 직렬화하고, 저장 시 처음 읽은 내용 해시와 비교해 오래된 초안의 덮어쓰기를 막아요.
- `maxExpandedBytes`로 클립보드·선택 영역·인코딩 변환 뒤 출력 크기를 제한해요.
- 규칙 저장 전 시간 스탬프 백업을 만들고 설정된 개수만 남겨요.

### RuleExchange

- UTF-8 BOM을 허용하고 쉼표 또는 탭 구분 파일을 같은 파서로 처리해요.
- 큰따옴표, 이중 큰따옴표 이스케이프, CRLF, 필드 내부 줄바꿈을 보존해요.
- 가져온 규칙은 기존 초안과 합친 뒤 RuleStore 전체 검증을 통과해야 반영돼요.
- 내보내기는 UTF-8 BOM 파일을 만들고 수식처럼 해석될 셀을 경고하거나 안전 접두사로 무력화해요.
- `extraJson` 열로 알려지지 않은 규칙 필드를 왕복 보존하며, 오류에는 행·열 위치를 포함해요.

### GroupManager

- 규칙 관리자 소유의 모달 Win32 창에서 그룹 ID, 표시 이름, 활성 상태를 편집해요.
- 그룹 ID 변경은 연결 규칙을 같은 초안 안에서 원자적으로 갱신해요.
- 그룹 삭제는 규칙을 지우지 않고 그룹 연결만 비운 뒤 전체 문서를 다시 검증해요.

### RuleManager

- 일반 Win32 모델리스 창이며 Notepad++에 `NPPM_MODELESSDIALOG`로 등록해요.
- 좌측 목록은 가상 ListView, 검색 디바운스, 그룹 필터, 정렬, 다중 선택 일괄 상태 변경을 제공해요.
- CSV/TSV 가져오기는 추가 또는 전체 교체를 선택할 수 있지만 저장 전까지 디스크를 건드리지 않아요.
- 그룹 콤보에서 새 ID를 입력하면 활성화된 그룹을 만들고, 별도 그룹 관리 창에서 상세 편집해요.
- 우측 편집기는 실행 조건, 확장자, 경로 글롭, 언어, 여러 줄 치환, 설명을 초안으로 편집하고 동적 변수 미리보기를 제공해요.
- 모든 초안 변경은 RuleStore로 다시 검증하며 저장 전에 자동 백업해요.
- Notepad++ 다크 모드 메시지와 per-monitor DPI 변경을 따라가며 창 위치·크기를 화면 안으로 보정해 기억해요.

### Plugin

- 버퍼 ID별 확장자와 문서별 일시정지 상태를 관리해요.
- 파일 저장·이름 변경·활성화 때 캐시를 갱신하고 파일 닫힘 때 상태를 제거해요.
- 설정과 규칙 재로드를 하나의 메뉴 명령으로 묶어요.
- Notepad++ API 경계에서 예외를 잡아 호스트로 전파되지 않게 해요.
- 날짜·시간·파일·클립보드·선택 영역·UUID·줄/열 변수를 확장하고 첫 탭 정지점으로 커서를 옮겨요.

## 현재 제한

- 대소문자 미구분은 영문 ASCII에만 적용돼요.
- 붙여넣기 자동 치환은 `processPaste` 선택 사항이며 마지막 구분 트리거 하나만 처리해요.
- 다중 선택/커서는 안전 모드에서 건너뛰어요.
- 시간 제한 없는 표준 정규식은 UI 스레드를 멈출 수 있어 정규식 트리거를 지원하지 않아요.
- 탭 정지점은 첫 위치만 선택하며 다음 위치 순회 세션은 아직 없어요.
- 실제 Notepad++ 호스트의 IME·ANSI/DBCS 조합은 수동 검증이 추가로 필요해요.
